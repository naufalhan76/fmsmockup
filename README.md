# GPS Tracker - Fleet Management System

Real-time fleet monitoring dashboard with trip tracking, temperature monitoring, and incident management.

## Features
- Live trip monitoring kanban board
- Temperature trend charts with threshold alerts
- 6-step shipping status tracking
- ETA calculation with route visualization
- Incident detection and alerting
- Manual status override system
- Responsive floating detail panels

## Tech Stack
- React 19 + Vite (Frontend)
- Node.js (Mock Backend)
- Leaflet (Maps)
- Chart.js (Temperature charts)

## Run Locally
`ash
npm install
npm run build
npm start
`
Open http://localhost:3000
